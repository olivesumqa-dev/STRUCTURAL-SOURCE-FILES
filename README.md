# STRUCTURAL-SOURCE-FILES

This repository contains reference source files for structural engineering apps under StrucForge.

## Purpose

These files are intended to guide the development of structural engineering calculation tools, including:

- RC beam design
- RC column design
- RC footing/foundation design
- structural analysis
- wind load calculation
- seismic load calculation
- load combinations
- structural drawing and detailing logic

## Instructions for Codex

1. Read this README first.
2. Use the PDF files as reference material only.
3. Do not copy long copyrighted text from any PDF.
4. Extract only formulas, design steps, assumptions, input requirements, limitations, and safety checks.
5. Create summarized Markdown notes inside the `/notes` folder.
6. When a requirement is unclear, mark it as `Needs engineer confirmation`.
7. Do not make unsafe assumptions.
8. Any structural app created from these sources must include warnings and must require final verification by a licensed structural engineer.

## Preferred Output

Create topic-based notes such as:

- `notes/rc-beam-design.md`
- `notes/rc-column-design.md`
- `notes/foundation-design.md`
- `notes/wind-loads.md`
- `notes/seismic-loads.md`
- `notes/load-combinations.md`
- `notes/structural-analysis.md`
